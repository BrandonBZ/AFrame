HTML - Hyper Text Markup Language

Elements

HTML is made of elements and attributes. HTML is for stucture of websites. CSS is for styling. JS is for functionality. Elements are comprised of an opening bracket, then a tag name, then a closing bracket, in between is the content, and finally it's the closing tag. Always indent for new elements.

Attributes

Attributes are qualities that descibe or define elements. Id, class, src, hrep, name, type are all attruibutes. Attributes are NOT ever in the closing tag. Ex: id="first", id is name, first is the value.
